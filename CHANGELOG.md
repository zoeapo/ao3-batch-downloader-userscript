# 更新记录

## 0.3.0

- 增加 Chrome 本地文件夹选择
- 使用 File System Access API 直接写入 EPUB 和 PDF
- 使用 IndexedDB 记住已选择的文件夹句柄
- 页面刷新后支持重新授权
- 同名文件自动增加编号，避免覆盖
- 增加 GitHub 自动更新地址
- 补齐 GitHub 项目文档、检查脚本和 Issue 模板

## 0.2.2

- 作品间隔改为随机范围
- 格式间隔改为随机范围
- 失败重试间隔改为随机范围
- 日志显示每次实际等待秒数
- 文件下载失败也会按随机间隔重试

## 0.2.1

- 修复 EPUB 被 Tampermonkey `not_whitelisted` 拦截的问题
- 增加二进制备用下载
- 加宽面板
- 支持拖动面板右下角调整大小

## 0.2.0

- 输入方式改为本地 TXT 文件
- 自动清洗和去重作品网址
- 增加失败链接、无效行和记录导出

## 0.1.0

- 初始实验版本
